## CardioBot

CardioBot é um chatbot para Telegram capaz de estimar o risco cardiovascular de um paciente após coletar certas informações. O sistema do chatbot integra um Large Language Model (LLM) como interface conversacional e um modelo de Machine Learning supervisionado como núcleo de decisão. A partir de respostas em texto livre do usuário, o LLM conduz uma conversa natural e acolhedora, extraindo informações clínicas relevantes (como idade, sexo, tabagismo, pressão arterial, colesterol e diabetes) e estruturando esses dados no formato esperado pelo modelo preditivo. A estimativa de risco é realizada exclusivamente por uma rede neural Multilayer Perceptron treinada com dados do Framingham Heart Study, um dos estudos epidemiológicos mais utilizados em pesquisas sobre risco cardiovascular. O CardioBot classifica o risco estimado em categorias (baixo, moderado ou alto) e fornece uma explicação em linguagem simples, reforçando que o resultado tem caráter educativo e não substitui a avaliação médica profissional.
